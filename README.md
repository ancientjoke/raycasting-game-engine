# Raycasting Game Engine
## Movement Controls
- `W` or `↑` - Move forward
- `S` or `↓` - Move backward
- `A` or `←` - Rotate left
- `D` or `→` - Rotate right
- `Q` - Strafe left
- `E` - Strafe right

## Game Controls
- `R` - Generate new map
- `G` - Toggle God Mode
- `M` - Toggle Minimap
- `SPACE` - Pause Enemy Movement

## Level Editor Controls
- `TAB` - Toggle Level Editor
- `1` - Select Wall Tile
- `2` - Select Empty Tile
- `3` - Toggle Enemy Placement
- `[` - Decrease Brush Size
- `]` - Increase Brush Size
- `S` - Save Map
- `L` - Load Map
- `DELETE` - Remove Enemy (when hovering over one)

- Press `R` to restart after death
